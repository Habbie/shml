# Emojis

To add an emoji you need to call the `emoji` method followed by an argument.

|---	|---	|---	|

### Code Samples

```bash
echo "$(color yellow 'My name is Dirt') $(emoji dog)"
```

### Results


`My name is Dirt 🐶`
